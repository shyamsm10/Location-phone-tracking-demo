# 📍 Location Phone Tracking Demo

## 🚀 Overview

This project is a **Location Phone Tracking Demo** that demonstrates how to track and display a device's location using modern web technologies. It is designed for learning purposes and showcases real-time location handling and basic tracking concepts.

---

## ✨ Features

* 📡 Real-time location tracking
* 🌍 Displays latitude and longitude
* 🖥️ Simple and user-friendly interface
* ⚡ Lightweight and fast
* 🔐 Works with browser location permissions

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend (if used): Python / Flask
* APIs: Browser Geolocation API

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/shyamsm10/Location-phone-tracking-demo.git
cd Location-phone-tracking-demo
```

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

## 🌐 Usage

1. Open the app in your browser
2. Allow location access
